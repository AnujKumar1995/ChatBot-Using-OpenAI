================================================================================
                          🔑 GETTING YOUR API KEY
================================================================================

1. Go to: https://platform.openai.com/api-keys
2. Sign up or log in with your GitHub/Google account
3. Click "Create new secret key"
4. Copy the key (starts with "sk-")
5. Add it to your .env file:
   OPENAI_API_KEY=sk-...

================================================================================
                       ⚙️ CONFIGURATION OPTIONS
================================================================================

Edit .env to customize:

OPENAI_API_KEY
   Your OpenAI API key (REQUIRED)

OPENAI_MODEL
   Model to use: gpt-3.5-turbo (default) or gpt-4

DEBUG
   Enable debug mode: True or False (default: False)

MAX_TOKENS
   Maximum response tokens: 1-4000 (default: 2000)

TEMPERATURE
   Response creativity: 0-2 (default: 0.7)
   0 = predictable, 2 = very random

HOST
   Server address (default: 0.0.0.0)

PORT
   Server port (default: 8000)

                          🚨 TROUBLESHOOTING
================================================================================

Q: "OPENAI_API_KEY is not set"
A: Make sure .env exists with your API key
   cp .env.example .env
   # Edit .env and add your OpenAI API key

Q: "Invalid OpenAI API key"
A: Check your key at https://platform.openai.com/api-keys

Q: "Cannot connect to server"
A: Make sure server is running:
   python -m uvicorn app.main:app --reload

Q: "Module not found" errors
A: Install dependencies:
   pip install -r requirements.txt

Q: "Rate limit exceeded"
A: OpenAI API has rate limits. Wait and try again.
